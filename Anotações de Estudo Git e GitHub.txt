Comandos básicos:

ls = listar diretórios
cd / = ir para diretório
cd .. = voltar para diretório anterior
clear ou ctrl L = limpar terminal
mkdir = criar diretório
echo = criar arquivo
del = deletar arquivo dentro de uma pasta
rmdir /S /Q = deletar pasta/diretório

Gerando encriptação = criando SHA1

Comando

open ssl sha1 "nome do arquivo"
